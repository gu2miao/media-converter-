# 媒体文件提取与格式转换器

这是一个基于网页的媒体文件处理工具，支持从各种网站提取视频文件以及进行多种媒体格式转换。

## 功能特性

- **视频下载**: 从B站、YouTube等视频网站提取视频
- **图片格式转换**: JPG、PNG、WebP、GIF等格式间互相转换
- **视频格式转换**: MP4、AVI、MOV、MKV、WebM等格式转换
- **音频格式转换**: MP3、WAV、OGG、FLAC等格式转换
- **图片转Base64**: 将图片转换为Base64编码
- **多种上传方式**: 支持拖拽上传、点击上传、剪贴板上传

## 技术架构

项目采用模块化设计，主要包含以下模块：

- `main.js`: 主应用逻辑
- `videoExtractor.js`: 视频提取模块
- `formatConverter.js`: 格式转换模块
- `fileUploader.js`: 文件上传处理模块
- `style.css`: 样式文件
- `yt-dlp.exe`:提供视频提取功能的模块（来自于https://github.com/yt-dlp/yt-dlp项目）

## 使用方法

1. 打开 `index.html` 文件即可使用
2. 通过左侧导航栏选择功能
3. 在中央区域进行具体操作

## 注意事项

- 当前版本为前端演示版本，视频提取功能需要后端服务支持
- 格式转换功能在浏览器端受限，复杂转换需要后端服务
- 请遵守相关网站的使用条款和版权法律

## 扩展开发

要将此工具扩展为完整版本，需要添加后端服务来处理：

- 视频提取和下载
- 复杂的格式转换
- 大文件处理
